1) устанавливаем пакет nodejs
npm init 

или 3в1> npm install --save express mongodb body-parser

2) устанавливаем сервер для nodejs
npm i --save express 

3) пакет который ищет ошибки кода и помогает превести код к общепринятому стилю
npm i --save-dev eslint

настройка:
1) ./node_modules/.bin/eslint --init
проверка на наличие ошибок кода
./node_modules/.bin/eslint .
в package.json пишем скрипт
"scripts": {
    "start": "nodemon main.js",
    "lint-me": "eslint .",
    "test": "echo \"Error: no test specified\" && exit 1"
  },
запуск проверки> npm start
4) пакет для подключения к MomgoDB
npm i --save mongoose

5)простой маленький пакет, который, при изменении файлов, автоматически перезапускает сервер:
> npm install --save-dev nodemon
настройка:
// package.json
  "scripts": {
    "start": "nodemon index.js",
  },

6)Для подключения и работы с базой данных в MongoDB:
> npm i --save mongoose
